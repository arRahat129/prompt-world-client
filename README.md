## 🐾 PromtWorld

## 🔍 Project Description

**PromtWorld** is a modern, community-driven pet adoption ecosystem engineered to connect companion animals with their ideal homes. Built to remove friction from the adoption lifecycle, the platform serves a dual purpose: enabling pet owners to securely list animals for adoption while providing prospective adopters with an intuitive, highly filterable discovery engine.

Designed with a focus on seamless user experience, PromptWorld integrates secure user authentication, personalized profile dashboards, and real-time adoption request workflows. The interface prioritizes responsiveness and fluid navigation, transforming a traditionally complex process into an accessible, digital-first community platform.

## 🛠️ Technologies to Use

| Technology | Purpose |
|---|---|
| **Next.js** | Build the UI and manage optimized server-side rendering and actions |
| **App router(Next.js)** | Handle page navigation, nested layouts, and dynamic route matrices |
| **Tailwind CSS+ HeroUI component library** | Styling and responsiveness with accessible, dynamic component architectures |
| **Framer Motion** | Power fluid page transitions and layout state animations |
| **Authentication** | BetterAuth (configured with MongoDB Adapter) for session verification |
| **MongoDB** | Handle primary data storage, connections, and aggregation pipelines |
| **Stripe Engine** | Process secure checkout flows, payment actions, and gateway verification |
| **Recharts** | Render interactive data visualization layers within dashboard interfaces |


## Key Features of this Project

* Advanced Search, Filter & Server-side Sorting Engine
* Role-Based Access Control & JWT Guardrails
* Premium Model Content Paywalling via Stripe
* Interactive Engineering Control Features
* Dynamic Data Aggregation Dashboards
* Automated Admin Moderation Lifecycle Pipeline
* Fluid Presentation Components Driven by Framer Motion
* Dual-state bookmarking engine with immediate database validation preventing asset duplication.
* Rejection feedback delivery architecture allowing admins to log specific revision reasons for pending creations.
* Unified global theme state implementing a reactive light and dark mode presentation framework.
